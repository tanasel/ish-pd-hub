# Adding & editing resources вҖ” a 1-page guide for leaders

The easiest way to add an opportunity is the **вҖңAdd a resourceвҖқ button** on the Hub itself
(top-right, or in the blue panel near the bottom). You never have to open a spreadsheet.

> You do **not** need to be technical. If you can fill in a short form, you can keep the Hub
> up to date.

---

## To add a new opportunity (the easy way)

1. Click **вҖңAdd a resourceвҖқ** on the Hub.
2. Enter the **staff access code** (only selected leaders have it вҖ” ask whoever set up the Hub).
3. Fill the short form. Only **Title** and **Category** are required вҖ” the rest just make the
   card richer. Add a **Date** for one-off events (a conference, a deadline) and the card
   tidies itself away automatically once that date has passed.
4. Click **Add resource**. It appears straight away and is saved for the whole team.

That's it вҖ” no spreadsheet needed.

## To edit or remove one

Added items are stored on the school's own server. To **edit or remove** one, ask whoever set
up the Hub вҖ” it's a quick change to the data file on the server. Adding new ones never needs this.

## What each form field means

| Field | What to write | Example |
|---|---|---|
| **Title** | The name of the opportunity | `Google Certified Educator Level 1` |
| **Category** | Pick **one** from the list below (spelling must match) | `EdTech & AI` |
| **Provider** | Who runs it | `Google for Education` |
| **Format** | How itвҖҷs delivered | `Online (self-paced)` |
| **Audience** | Who itвҖҷs for | `All staff` |
| **Cost** | Roughly what it costs | `Free` |
| **Description** | One sentence on what it is / why itвҖҷs useful | `Entry-level certification for Google tools.` |
| **URL** | The official link (leave blank for in-house) | `https://edu.google.com/...` |
| **Featured** | Type `Yes` to pin it near the top, otherwise leave blank | `Yes` |
| **Location** | Where it happens, or "Online" | `Online` В· `The Hague, NL` |
| **Date** | One-off events only вҖ” the card auto-archives after this date | `2026-10-22` |

## Good to know

- **Order doesnвҖҷt matter** вҖ” the Hub groups and sorts cards automatically; featured items rise to the top.
- **Leave a field blank** if it doesnвҖҷt apply (e.g. no link for an in-house session). The card adapts.
- **In-house sessions** (no link) show вҖңIn-house вҖ” ask your line managerвҖқ instead of a website button.
- **Anyone can browse**; only staff with the access code can add вҖ” so share the code only with the leaders you choose.
